Other libraries

The Ocamljs library gives declarations for the special primitives recognized by the external function mechanism. Depending on how you use them you may want to declare them with different types.

The Javascript library wraps a little bit of the standard Javascript library.

The Mozilla library wraps a little bit of the Mozilla XPCOM API.
