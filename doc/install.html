Installing ocamljs

  1. Install OCaml 3.10.0.
  2. Unpack the OCaml 3.10.0 distribution and run ./configure only, with the same options you gave when you built OCaml, then symlink it to ocamljs-0.1/ocaml.
  3. 'make' builds ocamljs and the standard library
  4. 'make test' runs the test suite; you will need SpiderMonkey or other Javascript interpreter
  5. 'make install' installs ocamljs and the standard library

Using ocamljs

The command 'ocamljs' may be used just like ocamlc or ocamlopt. It generates .cmj/.cmja files instead of .cmo/.cma. A linked file consists of Javascript source, which may be run through a standalone Javascript interpreter, or embedded in a web page or Firefox extension.

See myocamlbuild.ml for some ocamlbuild rules you may find useful.
